### Ceiling a number
Return value is float

    >>>import math
    >>>math.ceil(10.000001)
    >>>11.0
    >>>math.ceil(1)
    >>>1.0

### Floor a number
Return value is float

    >>>import math
    >>>math.floor(9.999999)
    >>>9.0
    >>>math.floor(8)
    >>>8.0

### Get the Square root of a number
Return value is float

    >>>import math
    >>>math.sqrt(9)
    >>>3.0
    >>>math.sqrt(3)
    >>>1.7320508075688772
